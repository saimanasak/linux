#### file_content_compare.sh  
- Files: *file1.txt*, *file2.txt*, *output.txt*
- This script is for two files.
- Checks whether the files are readable.
- Checks whether the files are empty.
- Checks both the files and if there's any matched content it prints the matched content to an output file.
- In simpler way, it checks for the data of the first file in the second file.
